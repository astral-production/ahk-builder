How to use AHK Builder

  1. Drag blocks from the left toolbox into the workspace
  2. Snap blocks together to build your script logic
  3. Click on numbers/text to edit values
  4. Right-click blocks to duplicate or delete
  5. Click Run to execute with AutoHotkey
  6. Click Export .ahk to save as a script file
